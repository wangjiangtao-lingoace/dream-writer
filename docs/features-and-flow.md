# Dream Writer 功能与流程详细文档

## 一、系统概述

Dream Writer 是一个 AI 驱动的长篇小说创作系统，采用"分层创作"理念，从灵感→卷纲→章纲→正文，逐层细化，确保长篇小说的稳定性和一致性。

---

## 二、核心功能模块

### 1. 小说管理

**功能：**
- 创建小说（标题、类型、灵感）
- 查询小说列表
- 查看小说详情（含章节、人物、资产）
- 更新小说信息
- 删除小说

**流程：**
```
用户填写小说信息 → 创建小说 → 自动生成第一章（空）
```

---

### 2. 人物管理

**功能：**
- 创建人物卡（名称、角色、身份、动机、外貌、背景、关系）
- 查询人物列表
- 编辑人物信息
- 删除人物

**流程：**
```
进入人物页面 → 填写人物信息 → 保存 → 人物列表展示
```

---

### 3. 世界观管理

**功能：**
- 创建世界观（名称、规则、地理、势力、历史）
- 查询世界观列表
- 编辑世界观
- 删除世界观

**流程：**
```
进入世界观页面 → 填写设定信息 → 保存 → 世界观列表展示
```

---

### 4. 卷纲系统

**功能：**
- 创建卷纲（卷名、目标、冲突、情绪、地图、钩子）
- AI 自动生成卷纲
- 查询卷纲列表
- 编辑卷纲
- 删除卷纲

**流程：**
```
方式一：手动创建
进入卷纲页面 → 点击"新建卷" → 填写信息 → 保存

方式二：AI 生成
进入卷纲页面 → 点击"AI 生成卷纲" → 设置卷数 → AI 生成 → 自动创建到数据库
```

---

### 5. 章纲系统

**功能：**
- 创建章纲（章节名、目标、冲突、情绪、爽点、钩子、伏笔）
- AI 自动生成章纲
- 查询章纲列表
- 编辑章纲
- 删除章纲

**流程：**
```
方式一：手动创建
选择卷 → 点击"新建章" → 填写信息 → 保存

方式二：AI 生成
选择卷 → 点击"AI 生成章纲" → 设置章数 → AI 生成 → 自动创建到数据库
```

---

### 6. 章节管理

**功能：**
- 创建章节
- 编辑章节（标题、目标、正文）
- 删除章节
- AI 生成正文
- AI 续写正文

**流程：**
```
创建章节 → 填写章节目标 → 点击"生成草稿" → AI 生成正文 → 保存

续写流程：
选择已有内容的章节 → 点击"生成草稿" → AI 基于已有内容续写 → 保存
```

---

### 7. 记忆系统

**功能：**
- 创建记忆（类型、分类、标题、内容、重要程度）
- 查询记忆列表（按类型筛选）
- 搜索记忆（关键词搜索）
- 记忆统计（按类型统计数量）
- 编辑记忆
- 删除记忆
- 批量创建记忆

**记忆类型：**
| 类型 | 说明 | 图标 |
|------|------|------|
| world | 世界记忆（设定、地图、势力） | 🌍 |
| character | 角色记忆（人设、关系） | 👤 |
| plot | 剧情记忆（已发生事件） | 📖 |
| foreshadow | 伏笔记忆（埋设/回收） | 🔗 |
| pleasure | 爽点记忆（避免重复） | ⚡ |
| style | 风格记忆（写作风格） | ✍️ |

**流程：**
```
进入记忆页面 → 选择类型 → 填写信息 → 保存 → 记忆列表展示

搜索流程：
输入关键词 → 点击搜索 → 展示匹配结果
```

---

### 8. 风格控制

**功能：**
- 创建风格配置
- 设置默认风格
- 编辑风格配置
- 删除风格配置
- 去 AI 味处理

**风格参数：**
| 参数 | 选项 | 说明 |
|------|------|------|
| 叙事视角 | 第一人称/第三人称/混合 | 控制叙述角度 |
| 时态 | 过去时/现在时 | 控制时态 |
| 节奏 | 慢/适中/快 | 控制剧情节奏 |
| 句子长度 | 短句/中等/长句/混合 | 控制句式 |
| 词汇风格 | 现代/古典/混合 | 控制用词 |
| 对话比例 | 少/适中/多 | 控制对话密度 |
| 情感强度 | 克制/适中/强烈 | 控制情感表达 |
| 幽默程度 | 无/少/适中/多 | 控制幽默元素 |

**特殊开关：**
- 避免 AI 味词汇
- 多用短句
- 多用对话
- 感官描写

**流程：**
```
创建风格 → 配置参数 → 设为默认 → AI 生成时自动应用

去 AI 味流程：
输入文本 → 点击"测试去 AI 味" → 查看处理结果
```

---

### 9. 拆书分析

**功能：**
- 创建拆书任务（标题、来源、原文）
- AI 自动拆书（8 维分析）
- 查看拆书结果
- 重建拆书
- 发布到知识库

**拆书维度：**
1. 拆书总览
2. 剧情结构
3. 故事时间线
4. 人物系统
5. 世界观与设定
6. 主题表达
7. 文风与技法
8. 商业化卖点

**流程：**
```
粘贴原文（≥80字） → 点击"创建拆书" → AI 分析 → 查看 8 维结果 → 发布到知识库
```

---

### 10. 知识库

**功能：**
- 创建知识资产（标题、分类、内容、标签）
- 查询知识资产列表
- 搜索知识资产
- 编辑知识资产
- 删除知识资产

**分类：**
- 设定
- 灵感
- 大纲
- 笔记
- 参考
- 其他

**流程：**
```
进入知识库 → 点击"入库新设定" → 填写信息 → 保存

拆书发布流程：
拆书完成 → 点击"发布到知识库" → 自动创建知识资产
```

---

### 11. 一致性校验

**功能：**
- 选择章节进行校验
- AI 自动检查 5 项内容
- 显示校验评分
- 展示问题列表和修复建议

**检查项目：**
| 类型 | 说明 |
|------|------|
| 战力 | 战力系统是否崩坏 |
| 人设 | 角色行为是否符合人设 |
| 世界观 | 设定是否前后矛盾 |
| 时间线 | 事件顺序是否正确 |
| 伏笔 | 伏笔是否遗忘 |

**流程：**
```
选择章节 → 点击"开始校验" → AI 分析 → 查看评分和问题列表
```

## 三、AI 生成功能

### 1. 灵感生成

**输入：** 类型、关键词
**输出：** 一句话 hook、高概念描述、核心爽点、读者画像、市场评估

### 2. 卷纲生成

**输入：** 小说信息、记忆上下文
**输出：** 多卷规划（卷名、目标、冲突、情绪、地图、钩子）

### 3. 章纲生成

**输入：** 卷纲信息、记忆上下文
**输出：** 多章规划（章节名、目标、冲突、情绪、爽点、钩子）

### 4. 正文生成

**输入：** 章纲信息、记忆上下文、风格配置
**输出：** 2000-3000 字正文

**特点：**
- 自动加载相关记忆
- 应用风格配置
- 去 AI 味处理
- 多用短句和对话

### 5. 正文续写

**输入：** 已有正文、章纲信息、记忆上下文
**输出：** 续写内容（1000-2000 字）

**特点：**
- 基于已有内容续写
- 保持风格一致性
- 自动去 AI 味

### 6. 一致性校验

**输入：** 章节内容、记忆上下文
**输出：** 5 项检查结果、评分、问题列表、修复建议

## 四、完整创作流程

### 标准流程

```
1. 创建小说
   ↓
2. 创建人物（主角、配角、反派）
   ↓
3. 创建世界观（规则、地理、势力）
   ↓
4. 添加记忆（世界设定、角色设定）
   ↓
5. 创建风格配置
   ↓
6. AI 生成卷纲
   ↓
7. AI 生成章纲
   ↓
8. 创建章节
   ↓
9. 填写章节目标
   ↓
10. AI 生成正文
    ↓
11. 一致性校验
    ↓
12. 保存 / 续写
```

### 拆书学习流程

```
1. 粘贴参考文本
   ↓
2. AI 拆书分析（8 维）
   ↓
3. 查看分析结果
   ↓
4. 发布到知识库
   ↓
5. 创作时参考
```

## 六、API 接口清单

## 七、数据模型

### 核心实体关系

```
Novel (小说)
├── Chapter (章节)
├── Character (人物)
├── Volume (卷纲)
│   └── ChapterOutline (章纲)
├── Memory (记忆)
├── StyleProfile (风格)
├── Foreshadow (伏笔)
├── PleasurePoint (爽点)
├── ConsistencyIssue (一致性问题)
└── KnowledgeAsset (知识资产)
```
