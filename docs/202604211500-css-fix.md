## 🔧 问题诊断

### 错误信息
```
[plugin:vite:import-analysis] Failed to resolve import "./ancient-theme.css" from "src/components/AncientPaper.tsx". Does the file exist?
```

### 问题根源
多个文件中CSS导入路径不正确：
- `AncientPaper.tsx` 导入 `./ancient-theme.css` (错误)
- `AncientHomePage.tsx` 导入 `./ancient-theme.css` (错误)
- `AncientWorkspace.tsx` 导入 `./ancient-theme.css` (错误)

实际的CSS文件位置：`src/styles/ancient-theme.css`

---

## 🛠️ 修复方案

### 1. 修复AncientPaper.tsx
**错误路径**: `import './ancient-theme.css'`
**正确路径**: `import '../styles/ancient-theme.css'`

### 2. 修复AncientHomePage.tsx
**错误路径**: `import './ancient-theme.css'`
**正确路径**: `import '../../styles/ancient-theme.css'`
**额外修复**: 添加了 `import './AncientHomePage.css'` (页面特定样式)

### 3. 修复AncientWorkspace.tsx
**错误路径**: `import './ancient-theme.css'`
**正确路径**: `import '../../styles/ancient-theme.css'`
**额外修复**: 添加了 `import './AncientWorkspace.css'` (页面特定样式)

## 💡 技术细节

### 相对路径解析
- **组件层** (`src/components/`): 需要 `../styles/` 引用样式目录
- **页面层** (`src/pages/ancient/`): 需要 `../../styles/` 引用样式目录
- **页面特定样式**: 同目录内相对路径 `./filename.css`

### 导入优先级
1. **全局主题**: `ancient-theme.css` (基础样式和变量)
2. **页面样式**: `AncientHomePage.css`、`AncientWorkspace.css` (页面特定)
3. **组件样式**: 内联样式或CSS模块（如需要）
