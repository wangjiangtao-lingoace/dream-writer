### 2.3 角色标签深化

#### 林凡（男主）
**现有标签**：普通起点、不认命、善良、有底线
**深化标签**：
- 核心矛盾：普通人 vs 非凡命运
- 成长弧：丧怂抱怨 → 自信有担当 → 从容沉稳
- 行为红线：不会主动英雄救美、不会一夜变强、不会放弃普通人身份认同
- 语言风格：前期吐槽抱怨、中期自信坚定、后期从容淡定
- 禁止行为：突然变圣父、突然变装逼、突然变无所不能

#### 林富贵（老祖）
**现有标签**：怂但不坏、阴但不毒、装但不崩
**深化标签**：
- 核心矛盾：想装逼 vs 实际怂
- 行为模式：吹牛 → 遇怂 → 偷袭 → 狐假虎威
- BGM癖：戏曲选段、唢呐曲、影视配乐
- 绝不会做的事：正面单挑、承认自己怂、背叛林凡
- 语言风格：老派、训人、吹嘘、阴阳怪气
- 禁止行为：突然变英勇、突然变真诚、突然变正经

#### 陆清菲（女主）
**现有标签**：温柔但不软弱、平凡但不平庸
**深化标签**：
- 核心矛盾：普通人视角 vs 灵异世界介入
- 行为模式：观察 → 共鸣 → 分寸 → 间接帮助
- 感情线：初遇(1-20) → 好感(21-50) → 暧昧(51-80) → 稳定(81+)
- 绝不会做的事：变成战斗型女主、突然变推理型、矫情做作
- 语言风格：温柔、细致、有分寸
- 禁止行为：突然变强势、突然变依赖、突然变花瓶

#### 王德发（死党）
**现有标签**：沙雕但不傻、添乱但不坏
**深化标签**：
- 核心功能：喜剧担当、兄弟情锚点、普通人视角对照
- 行为模式：闯祸 → 搞笑 → 关键时刻靠得住
- 出场频率：前期每3-5章、中期每5-8章、后期关键节点
- 绝不会做的事：变聪明、变正经、背叛兄弟
- 语言风格：沙雕、搞笑、接地气
- 禁止行为：突然变聪明、突然变严肃、突然变有用

#### 萧慕晴（女配）
**现有标签**：傲娇但不恶毒、优越但不刻薄
**深化标签**：
- 核心矛盾：傲娇 vs 真心
- 关系四阶段：无视 → 好奇 → 认可 → 独立平行线
- 傲娇指南：口是心非但有分寸，行动上不能贬低
- 绝不会做的事：恶意伤害林凡、成为感情线、突然倒贴
- 语言风格：傲慢、嘴硬、偶尔流露关心
- 禁止行为：突然变温柔、突然变依赖、突然变花痴

#### 钟少府（阴间同僚）
**现有标签**：武力强、策略弱
**深化标签**：
- 核心矛盾：能力强 vs 职场情商低
- 行为模式：直接 → 冲突 → 被利用 → 反击
- 语言风格：直率、不拐弯、偶尔吐槽
- 禁止行为：突然变圆滑、突然变阴险、突然变深情

### 4.2 章节处理方案

**方案A：保留现有章节，扩展后续**
- 保留现有 150 章（修复重复问题后）
- 从第 151 章开始扩展到 500 章
- 优点：工作量小，保留已有内容
- 缺点：前 150 章字数偏多（平均 3405 字）

**方案B：重新生成所有章节**
- 删除所有现有章节
- 按照新的 2500 字标准重新生成 500 章
- 优点：质量统一，符合新标准
- 缺点：工作量大，丢失已有内容

**方案C：拆分现有章节 + 扩展后续**
- 将现有 150 章拆分为约 200 章（每章 2500 字）
- 从第 201 章开始扩展到 500 章
- 优点：保留已有内容，符合新标准
- 缺点：拆分可能破坏原有节奏

**建议**：采用方案C
