# 变更记录 202606201700

## 变更原因
用户要求将 UI 从"古色古香"风格改为 Notion 风格极简 SaaS 设计。经审计，"古风"元素仅限于 LXGW WenKai 衬线字体，但整体 UI 有多处渐变、毛玻璃、glow 等非极简元素需要清理。

## 修改点

### 1. Token 系统重写（tokens.css）
- 颜色：冷灰 `#f5f7fa` → 暖白 `#ffffff`，暖灰 `#f7f6f3`
- 文字：`#1f2937` → `#37352f`（Notion 暖色文字）
- 强调色：`#4f7cff` → `#2383e2`（Notion 蓝）
- 阴影：全部扁平化，`--shadow-sm` 改为 `none`，移除 `--shadow-glow`
- 圆角：缩小（3/4/6/8px）
- `--font-serif` 改为指向 `--font-sans`
- 新增 `--accent-glow: none` 和 `--shadow-glow: none` 兼容别名

### 2. 移除衬线字体（6 个 TSX 文件，15 处）
- WorkspaceHeader、RichTextEditor、NovelForm、CreateWork、AnalyzeCreate、ConsistencyPanel
- 所有 `fontFamily: "var(--font-serif)"` 已移除，统一使用 Inter

### 3. CSS 去渐变/去阴影（3 个 CSS 文件，14 处）
- components.css：按钮去渐变、输入框去 glow ring
- bookshelf.css：banner/图标/进度条去渐变
- workspace.css：章节激活态边框柔化

### 4. TSX 内联样式清理（10+ 个文件）
- 移除所有 `backdropFilter: "blur()"` 效果
- 模态框遮罩统一为 `rgba(15,15,15,0.6)`
- 模态框内容背景改为 `var(--bg-surface)`
- 进度条/品牌图标去渐变
- 硬编码旧主题色全部替换为 CSS 变量

### 5. 字体导入清理（base.css）
- 移除 LXGW WenKai Google Fonts 导入
- 滚动条颜色适配暖色调
- ::selection 颜色适配新强调色

### 6. 装饰性背景移除
- NovelForm.tsx：移除 SVG 网格背景图案
- CreateWork.tsx：移除 SVG 网格背景图案
