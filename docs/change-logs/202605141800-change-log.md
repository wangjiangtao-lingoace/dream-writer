# Dream Writer 工作台 UI 优化

## 变更原因
用户反馈工作台存在多个UI问题，需要逐一修复以提升用户体验。

## 修改点

### 1. 移除顶部章节目录
- 移除顶部章节目录可折叠面板，避免与左侧章节列表功能重复
- 清理不再使用的状态变量 `expandedChapterId` 和 `chapterPanelCollapsed`

### 2. 写作编辑框优化
- 加大编辑框内边距：`padding: 2rem 3rem`
- 增加行高：`line-height: 2`
- 设置最大宽度：`max-width: 900px`，居中显示
- 字体大小调整为 `1rem`

### 3. 章节来源翻译扩展
- 添加 `imitation_pipeline` 翻译为"仿写流水线"
- 添加 `pipeline` 翻译为"流水线"

### 4. 英文状态显示翻译
- 总控面板中 `adoption` 的 key 和 value 使用翻译函数
- 成果与使用记录中 `assetType` 使用 `translateAssetType` 翻译

## 文件列表

| 文件 | 变更说明 |
|------|----------|
| `client/src/pages/NovelWorkspace.tsx` | 移除顶部章节目录，清理状态变量 |
| `client/src/utils/translate.ts` | 扩展章节来源翻译 |
| `client/src/styles/pages/workspace.css` | 优化编辑框样式 |

### 5. 写作编辑框高度修复
- 设置 `.write-panel` 最小高度为 `calc(100vh - 200px)`
- 确保 `.write-content` 正确使用 flex 布局
- 编辑框现在会自动占满可用空间

## 验证情况

1. TypeScript 类型检查通过
2. Vite 构建成功，无错误
3. 功能验证：
   - ✅ 顶部章节目录已移除
   - ✅ 左侧章节列表保留
   - ✅ 编辑框更大更舒适
   - ✅ 编辑框高度正确占满可用空间
   - ✅ 章节来源正确翻译
