## 修改点

### 修改文件
- `server/src/routes/search.ts` — 扩展验证源域名，改进置信度评分
- `server/src/services/ContentExtractionService.ts` — 改进内容提取，修复正则安全问题

### 功能变更

1. **扩展验证源域名（3层分级体系）**
   - Tier 1（高权重正版站）：qidian.com, jjwxc.net, zongheng.com, 17k.com, ciweimao.com, qimao.com, faloo.com, shuqi.com
   - Tier 2（资讯/社区站）：zhihu.com, douban.com, baike.baidu.com
   - Tier 3（抓取/聚合站）：guazixs.cn, lrts.me, fanqienovel.com, sogou.com
   - Tier 1 结果 +0.1 置信度，Tier 2 +0.05 置信度

2. **改进置信度评分**
   - 长度惩罚：<30字符 -5，>=3000字符 -5
   - 结构化内容奖励："作者"+"简介"+"主角" >= 2个 +15
   - 叙事动词奖励：每个 +2
   - 章节目录惩罚：>= 3章节模式 -20
   - 小说站点容器奖励：class="intro"/"bookinfo" +20

3. **改进内容提取（5级优先级链）**
   - Level 1: `<meta name="description">` 和 `<meta property="og:description">`
   - Level 2: 小说站点容器（`<div class="intro">`, `<div class="bookinfo">` 等10种模式）
   - Level 3: "简介"关键词块
   - Level 4: 包含标题的高分块
   - Level 5: 最高分块

4. **安全修复**
   - 修复 `extractNovelSiteSynopsis()` 正则灾难性回溯风险，改用 indexOf + 深度计数
   - 添加 50KB HTML 扫描限制
   - 修复域名 `includes()` 匹配漏洞，改为后缀精确匹配
