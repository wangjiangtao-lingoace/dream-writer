# 变更记录

## 变更原因
根据产品 review 反馈，将工作台 13 个平铺模块收敛为 5 大工作区，降低用户认知负担。写作页改为默认首页，产品定位从"全能创作平台"收敛为"AI 长篇创作工作台"。

## 修改点

### 新增类型定义
- `client/src/components/workspace/types.ts` — 新增 `WorkspaceGroupId` 类型和 `WorkspaceGroupDef` 接口
- `client/src/components/workspace/index.ts` — 导出新类型

### 侧边栏重写
- `client/src/components/workspace/WorkspaceSidebar.tsx` — 从平铺式 tab 列表改为 5 组手风琴导航，支持折叠态只显示图标，活动分组高亮，子 tab 缩进显示

### 工作台主逻辑重构
- `client/src/pages/NovelWorkspace.tsx`
  - 默认 tab 从 `dashboard` 改为 `write`
  - 新增 `TAB_TO_GROUP` 映射表（13 tab → 5 group）
  - 新增 `activeGroupId` 派生状态
  - 将 `tabGroups` 替换为 `groupDefs`（5 大工作区定义）
  - 新增 `handleGroupClick` 处理器
  - 新增 `sidebarCollapsed` 状态
  - 布局分支从 `activeTab === "write"` 改为 `activeGroupId === "writing"`
  - 非写作布局新增内容区子 tab 栏

### CSS 样式新增
- `client/src/styles/pages/workspace.css` — 新增 `.workspace-group-tabs`、`.group-tab`、`.workspace-content-inner` 样式

## 新的 5 大工作区

| 工作区 | 包含模块 | 默认首页 |
|--------|----------|----------|
| 正文写作 | 写作、拆书 | 是 |
| 创作总控 | 总控 | |
| 故事规划 | 大纲、卷纲、主线、钩子 | |
| 创作资产 | 人物、世界观、风格、知识库 | |
| 质量校验 | 记忆、校验 | |

## 风险说明
- 所有 13 个 tab key 保持为合法 URL 段，深链接不受影响
- 写作页三栏布局不变，只是从"tab"变为"默认首页"
- 现有组件（DashboardPanel、CharacterCard 等）全部复用，无修改
- 不涉及后端接口变更

## 验证情况
- TypeScript 全量编译通过（shared + server + client）
