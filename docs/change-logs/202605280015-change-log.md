# 变更记录

## 变更原因
UI/UX 现代化第四轮：统一设计令牌。将所有组件中残留的硬编码旧琥珀色值（rgba(249,115,22)、rgba(139,69,19)）替换为 CSS 变量，确保主题色统一。

## 修改点
将 ~25 处硬编码旧主题色替换为 CSS 变量：
- `rgba(249,115,22,0.05~0.1)` → `var(--accent-muted)`
- `rgba(139,69,19,0.03~0.08)` → `var(--accent-muted)`
- `rgba(249,115,22,0.15~0.2)` → `rgba(99,102,241,0.15~0.2)`
- ConsistencyPanel 严重度颜色：`#c62828`/`#e65100`/`#2e7d32` → `var(--error)`/`var(--warning)`/`var(--success)`
- ConsistencyPanel 评分背景：硬编码 rgba → `var(--success-muted)`/`var(--warning-muted)`/`var(--error-muted)`
