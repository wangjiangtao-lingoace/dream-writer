# 变更记录：功能引导页面

## 变更原因
用户反馈新用户不了解系统的工作流程（独立创作 vs 拆书仿写），需要在首页增加功能引导入口，帮助用户快速上手。同时需要建立 skill 机制，确保后续功能变更时同步更新引导内容。

## 修改点

### 新增功能引导页面
- 新增 `GuidePage` 组件，包含两种创作模式的详细步骤说明和使用技巧
- 独立创作模式：5 步流程（填写创意→生成大纲→审核大纲→生成资产→生成章节）
- 拆书仿写模式：5 步流程（输入书名→自动拆书→生成仿写方案→资产落库→自动创作）
- Tab 切换 UI，支持查看不同模式的引导

### 路由和导航集成
- 路由新增 `/guide` 路径
- 左侧菜单"工具"组新增"功能引导"入口（问号图标）

### Skill 机制
- 新增 `.superpowers/update-guide/skill.md`，定义触发条件和检查清单
- 当 Pipeline 流程、创作模式或核心功能变更时，自动提醒更新引导页面

## 风险说明
- 纯新增功能，不影响现有流程
- GuidePage 为静态内容页面，无 API 依赖
