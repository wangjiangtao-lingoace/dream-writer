## 设计原则
1. **简约现代** - 使用卡片式布局，留白充足，层次分明
2. **沉浸感** - 古风元素与现代UI结合，不杂乱
3. **清晰导航** - 扁平化导航，减少层级
4. **一致性** - 统一的按钮、卡片、输入框样式

## 样式特点
- 使用CSS变量系统，便于主题定制
- 卡片使用圆角、阴影，增加层次感
- 按钮使用渐变或实心样式，提升视觉吸引力
- 保持古风配色（米白、黛青、朱砂、墨黑）
- 添加平滑过渡动画，提升用户体验
- 使用SVG图标，保持清晰度和一致性
