# 章纲标题风格约束优化

## 变更原因

流水线生成的章节标题出现“祭祖也能被绑架？”“窥探镜卡顿如PPT”“我的第一个KPI：劝鬼回家”等偏章纲说明、偏段子化的标题，与用户原文前三章标题风格不一致。

## 修改点

1. 新增章节标题风格硬约束：
   - 标题必须像已发表小说目录，不得像章纲说明
   - 优先 2-8 个汉字，最多 12 个字
   - 禁止使用 PPT、KPI、系统、任务、规则、金手指、爽点、打工人、绩效 等设定说明词
   - 参考用户原文标题：《他们当时的嘲笑声好大呀》《上香》《第一个任务》

2. 将标题规则注入 `generateEnrichedChapterOutlines` prompt。

3. 增加回归测试，确保标题规则文本包含关键禁用词和原文参考标题。
