# 变更记录

## 变更原因
为 MemoryPanel 组件添加剧情状态可视化、记忆层级可视化、压缩操作和记忆统计功能。

## 修改点
1. **剧情状态可视化**
   - 调用 `GET /api/story-state/:novelId` 获取剧情状态
   - 展示当前卷、当前章、当前阶段
   - 展示主角状态（等级、目标、处境）
   - 展示情绪状态（当前情绪、强度、累积压抑值）
   - 展示爽点状态（上次爽点章节、冷却值）
   - 展示伏笔状态（活跃伏笔数、待回收数）

2. **记忆层级可视化**
   - 用不同颜色标识不同层级的记忆
   - 永久记忆（红色背景 #fee2e2）
   - 长期记忆（橙色背景 #ffedd5）
   - 短期记忆（黄色背景 #fef9c3）
   - 临时记忆（灰色背景 #f3f4f6）
   - 记忆卡片左侧边框使用对应层级颜色

3. **压缩操作**
   - 添加"压缩记忆"按钮，调用 `POST /api/memory-compression/:novelId/consolidate`
   - 添加"自动管理"按钮，调用 `POST /api/memory-compression/:novelId/auto-manage`
   - 操作过程中显示加载状态

4. **记忆统计**
   - 显示各层级记忆数量
   - 层级统计可点击筛选

## 文件列表
- `client/src/components/MemoryPanel.tsx` - 主要修改文件，添加新功能

## 风险说明
- 依赖后端 API `/api/story-state/:novelId` 和 `/api/memory-compression/:novelId/*` 接口
- 依赖 `/api/memories/:novelId/level-stats` 接口
- 如果后端接口不存在，相关功能将静默失败（仅 console.error）
