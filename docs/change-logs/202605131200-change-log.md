# 变更日志

## 变更原因
为前端组件添加AI生成功能，提升用户体验和创作效率。

## 修改点

### 1. MainlinePanel 组件增强
- 新增"AI生成主线"按钮
- 调用 `POST /api/ai/generate-mainlines` 接口
- 支持手动编辑和删除主线
- 添加加载状态和错误处理

### 2. HookPanel 组件增强
- 新增"AI批量预设钩子"按钮
- 调用 `POST /api/ai/generate-hooks` 接口
- 支持手动编辑和删除钩子
- 添加加载状态和错误处理

### 3. StylePanel 组件增强
- 新增"从文本学习风格"按钮
- 用户可粘贴参考文本（至少100字）
- 调用 `POST /api/ai/extract-style` 接口
- 显示提取的风格特征
- 支持将提取结果应用到表单

### 4. VolumeEditor 组件增强
- Volume 接口新增字段：`chapterCount`、`keyEvents`、`turningPoints`
- 卷纲表单添加这三个字段的输入
- 卷纲卡片显示新字段信息
- 支持编辑新字段

## 文件列表
- `/client/src/pages/NovelWorkspace.tsx` - 修改 MainlinePanel 和 HookPanel 组件
- `/client/src/components/StylePanel.tsx` - 添加从文本学习风格功能
- `/client/src/components/VolumeEditor.tsx` - 增强卷纲字段

## 风险说明
- 无破坏性变更，所有修改均为功能增强
- 新增功能依赖后端API接口，需确保接口可用
- TypeScript 类型检查已通过
