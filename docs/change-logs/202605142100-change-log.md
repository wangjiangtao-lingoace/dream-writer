# UI 全面重构变更记录

**变更原因**: 将 Dream Writer 从古风主题改造为深色科技风（炭黑琥珀），实现三栏布局 + AI 上下文操作面板，支持 BYOK（用户自带 API Key）。

**分支**: `feat/lightweight-rebuild`

## 修改点

### 1. Design Token 系统
- 新增 `tokens.css`: 深色科技风 CSS 变量（背景、边框、文字、主色琥珀橙、语义色、间距、圆角、阴影、字体、过渡）
- 新增 `base.css`: 全局基础样式、Google Fonts 导入、滚动条样式、关键帧动画
- 新增 `components.css`: 通用组件类（.btn、.input、.card、.badge、.skeleton、.empty-state）
- 保留向后兼容别名（--bg-primary → --bg-base、--bg-card → --bg-surface、--border → --border-default）

### 2. UI 组件库
- 新增 `Button.tsx`: 支持 variant（primary/secondary/ghost/danger）、size（sm/md/lg）、loading 状态
- 新增 `Card.tsx`: 支持 hoverable、onClick
- 新增 `Input.tsx`: forwardRef、label 和 error 显示
- 新增 `Modal.tsx`: 固定 overlay、ESC 关闭、背景模糊
- 新增 `Skeleton.tsx`: width/height/borderRadius/count
- 新增 `Tabs.tsx`: Tab 栏 + 活跃指示器

### 3. 布局组件
- 新增 `AppLayout.tsx`: 三栏布局容器（侧边栏 + 主内容 + AI 面板）
- 新增 `Sidebar.tsx`: 固定文字侧边栏（200px，可折叠），分组：项目/工具/设置
- 新增 `TopBar.tsx`: 顶部栏（48px），面包屑 + 操作槽
- 新增 `AIPanel.tsx`: 右侧 AI 面板（280px），上下文感知操作

### 4. AI Context 和 Hook
- 新增 `AIContext.tsx`: AI 面板状态管理（loading/result/execute/clearResult）
- 新增 `useAI.ts`: AI 操作 hook（generateContent/checkConsistency/streamContent），支持 AbortController 清理
- 新增 `useConfig.ts`: AI 配置 hooks（useAIConfigs/useDefaultConfig/useCreateConfig/useDeleteConfig/useTestConfig）

### 5. 页面重构
- **BookShelf.tsx**: 从古风书架改为深色列表视图，新增 API Key 配置横幅
- **NovelWorkspace.tsx**: 三栏布局（章节目录 | 编辑器 | AI 面板），上下文感知 AI 操作
- **PipelinePage.tsx**: 横向阶段流 + 章节列表可视化
- **Settings.tsx**: 新增设置页面，AI 模型配置管理（添加/删除/测试连接）

### 6. 路由更新
- 新增 `/settings` 路由
