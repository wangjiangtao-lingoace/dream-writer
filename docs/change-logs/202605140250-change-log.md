- **问题**: Anthropic case 直接返回 `null`，配置了但无法使用
- **修复**: 删除硬编码的 `return null`，恢复通用 provider 解析流程

### 5. 修复正文生成流式输出
- **文件**: `server/src/services/llm/LlmInvokeService.ts`, `server/src/services/AIService.ts`, `server/src/routes/ai.ts`
- **问题**: 正文生成使用"模拟流式"，先生成完整内容再分块发送

- **提交**: 已提交

- **文件**: `server/src/routes/batchGenerate.ts`
- **问题**: 批量生成使用串行循环，耗时极长
- **修复**: 实现分批并行执行，每批最多 3 章并发
- **提交**: `48899f4`

### 7. 修复自动生成状态持久化
- **文件**: `server/prisma/schema.prisma`, `server/src/services/AutoGenerateService.ts`
- **问题**: 全局状态存储在内存 Map 中，进程重启后丢失
- **修复**: 新增 `AutoGenerateStatus` 表，使用数据库持久化替代内存 Map
- **提交**: `49679ec`
