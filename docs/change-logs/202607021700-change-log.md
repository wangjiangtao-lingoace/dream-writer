# 变更记录 - 2026-07-02 17:00

## 变更原因
创建首次使用向导组件，提升新用户上手体验。

## 修改点
1. 新增 `OnboardingGuide` 组件，包含 3 步引导流程：
   - 第 1 步：欢迎 + 系统功能介绍
   - 第 2 步：选择创作方式（独立创作/拆书创作/导入续写）
   - 第 3 步：快速开始指引
2. 使用 localStorage 记录引导完成状态
3. 首次进入 BookShelf 页面时自动展示引导
4. 支持跳过引导和 ESC 键关闭
5. 样式与现有古风主题风格一致

## 文件列表
- `client/src/components/OnboardingGuide.tsx` - 新增，引导组件主体
- `client/src/styles/components/onboarding.css` - 新增，引导组件样式
- `client/src/pages/BookShelf.tsx` - 修改，集成引导组件

## 风险说明
- 引导状态存储在 localStorage，清除浏览器数据后会重新显示
- 不影响现有功能，仅在首次访问时展示

## 验证情况
- 类型检查通过（唯一错误为 NovelWorkspace.tsx 已存在问题）
- 组件逻辑完整：跳过、关闭、步骤切换、路由跳转均正常
