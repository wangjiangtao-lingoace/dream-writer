## 附：拆书数据源问题分析

### 问题根因
1. 数据源全部依赖硬编码站点（起点/纵横/17K/瓜子/LRTS）
2. 网页抓取受反爬机制影响，失败率高
3. Sogou 兜底能力弱，置信度仅 0.35-0.45
4. 不支持用户直接输入文本

### 升级路径
1. **阶段一**: 接入通用搜索 API + 支持文本粘贴
2. **阶段二**: RAG 管线（向量数据库 + 嵌入模型）
3. **阶段三**: 多轮迭代生成 + 风格校验
