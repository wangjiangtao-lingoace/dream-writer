## 风险说明
- `localDate()` 使用服务器本地时区，非 UTC。国内部署无问题，海外部署需注意
- 角色进度条使用 `arcSummary` 存在性判断，非精确百分比，后续可接入真实进度数据
- 世界观数据通过独立 API 加载（非 workspace-data），写 tab 切换时数据已预加载
