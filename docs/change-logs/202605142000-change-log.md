# 变更记录 2026-05-14 20:00

## 变更原因
前端 UI 修复工作流：替换原生弹窗、修复标签过滤、记忆显示截断、知识条目展开、风格翻译。

## 修改点

### 1. 替换原生弹窗（alert/confirm/prompt）
- NovelForm.tsx: `alert()` -> `toast.error()`
- VolumeEditor.tsx: `prompt()` -> 内联数字输入组件；`confirm()` -> `ConfirmDialog`
- GeneralKnowledge.tsx: `alert()` -> `toast.error()`；`confirm()` -> `ConfirmDialog`
- CharacterCard.tsx: `confirm()` -> `ConfirmDialog`
- MemoryPanel.tsx: `confirm()` -> `ConfirmDialog`
- StylePanel.tsx: `confirm()` -> `ConfirmDialog`

### 2. 人物标签过滤功能
- CharacterCard.tsx: 标签区域从 `<span>` 改为 `<button>`，点击可按角色类型过滤人物列表

### 3. 记忆重要程度显示修复
- MemoryPanel.tsx: 将 `"★".repeat(Math.min(importance, 5))` 替换为进度条 + 数字显示（x/10），importance 1-10 完整展示

### 4. 知识条目展开/收起
- GeneralKnowledge.tsx: 超过 200 字符的内容默认截断，添加"展开/收起"按钮

### 5. 风格提取结果中文翻译
- StylePanel.tsx: 新增 `translateStyleValue()` 统一翻译函数，覆盖 narrativePov、tense、pacing、sentenceLength、vocabulary、dialogueRatio、emotionIntensity、humorLevel 全部字段

### 6. ConfirmDialog 组件
- 新建 `client/src/components/ui/CommonComponents.tsx`，提供轻量 ConfirmDialog 组件
