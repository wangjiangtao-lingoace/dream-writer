# 变更记录

## 变更原因
UI/UX 现代化第一轮：主题色切换。将暖色琥珀系 (#f97316) 替换为冷色靛蓝系 (#6366f1)，字体从 KaiTi 楷体切换为 Inter 现代无衬线字体，为后续组件统一奠定基础。

## 修改点
1. **tokens.css**: 主色从 `#f97316`(amber) 改为 `#6366f1`(indigo)，更新所有 accent 变体（hover/muted/glow），font-serif 从 KaiTi 改为 Inter
2. **base.css**: Google Fonts 引入添加 Inter 字体
3. **bookshelf.css**: 2处硬编码 rgba(249,115,22) → rgba(99,102,241)
4. **settings.css**: 1处硬编码 rgba(249,115,22) → rgba(99,102,241)
5. **pipeline.css**: 1处硬编码 rgba(249,115,22) → rgba(99,102,241)

## 文件列表
- `client/src/styles/tokens.css` — 主题色和字体变量
- `client/src/styles/base.css` — Google Fonts 引入
- `client/src/styles/pages/bookshelf.css` — 书架页硬编码色值
- `client/src/styles/pages/settings.css` — 设置页硬编码色值
- `client/src/styles/pages/pipeline.css` — 流水线页硬编码色值

## 风险说明
- 纯视觉层改动，不影响任何接口或业务逻辑
- 所有页面通过 CSS 变量自动继承新主题
- 个别组件内联样式中的硬编码色值需后续轮次统一

- 书架、设置、流水线页面视觉正常
