# 古风UI样式更新

## 变更原因
根据产品需求，对前端页面进行古风样式改造，提升用户体验和视觉一致性。

## 修改点
1. 更新色彩系统为古风配色（米白、深棕、赤棕等）
2. 书架页面添加纸张纹理背景，书法体品牌Logo，古风卡片样式
3. 创建作品页面添加古风装饰边框和水墨元素装饰
4. 新建作品表单页面添加古风边框和宣纸纹理背景
5. 工作台页面左侧导航使用古风标签样式，内容区使用卡片式布局
6. 流程页面进度条使用古风装饰，阶段卡片使用线装书边框

## 文件列表
- `client/src/styles/variables.css` - 更新色彩系统变量
- `client/src/pages/BookShelf.tsx` - 书架页面古风样式改造
- `client/src/pages/CreateWork.tsx` - 创建作品页面古风样式改造
- `client/src/pages/NovelForm.tsx` - 新建作品表单古风样式改造
- `client/src/pages/NovelWorkspace.tsx` - 工作台页面古风样式改造
- `client/src/pages/PipelinePage.tsx` - 流程页面古风样式改造

## 风险说明
- 所有修改均为样式层面的内联样式更新，未修改业务逻辑
- 使用CSS变量确保主题一致性
- 保留原有功能，仅视觉呈现变化

## 验证情况
- TypeScript类型检查通过：`pnpm typecheck:client`
- 所有页面样式已更新，功能保持不变
