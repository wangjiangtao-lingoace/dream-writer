A. 首次启动弹窗

首次打开应用时弹出配置弹窗：选择提供商 → 选择模型 → 填入 API Key。可跳过稍后设置。之后在侧边栏设置页可随时修改。引导感强。

B. 侧边栏设置页

侧边栏底部「设置」入口，进入后管理所有 AI 提供商：添加/编辑/删除 API Key，切换默认模型。卡片式展示每个提供商状态。不需要弹窗打断流程。

C. 首页顶部引导横幅

书架首页顶部显示配置引导横幅（未配置时）。点击后弹出配置弹窗。配置完成后横幅消失。不打断流程，但有明确引导。两者结合：A 的弹窗 + C 的入口。
