# update-guide

当 Pipeline 流程、创作模式或核心功能发生变更时，检查并更新功能引导页面。

## 触发条件

完成以下类型的工作后，必须执行此 skill：

1. **Pipeline 流程变更** — 阶段增删、步骤调整、确认流程修改
2. **创作模式变更** — 独立创作或拆书仿写的步骤、交互方式变化
3. **核心功能变更** — 知识库、人物卡、世界观、风格配置等资产的生成或管理方式变化
4. **UI 交互变更** — 编辑、重新生成、确认等操作方式变化

## 检查清单

1. 读取 `client/src/pages/GuidePage.tsx`
2. 对比当前功能实现与引导描述是否一致
3. 如有差异，更新 GuidePage.tsx 中对应的：
   - `steps` 数组（步骤描述和详情）
   - `description`（模式描述）
   - `tips`（使用技巧）
   - 如有新模式，添加新的 section
4. 确保语言简洁、面向用户（非技术描述）

## 引导页面位置

`client/src/pages/GuidePage.tsx`

## 当前包含的模式

- **独立创作** (standalone) — 从灵感生成大纲，分步确认后生成资产和章节
- **拆书仿写** (imitation) — 输入书名，自动拆书分析后仿写创作
