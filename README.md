# Dream Writer - 古色古香AI小说创作助手

> "墨香流传，笔耕不辍"

一个具有传统线装书风格的AI小说创作平台，为写作者提供沉浸式的古风创作体验。

## 🎨 设计理念

### 视觉风格
- **传统线装书**: 米黄色宣纸背景，竖排文字暗示，古典边框装饰
- **文人书房感**: 木纹质感，毛笔字效，古典家具元素
- **水墨意境**: 淡雅色调，山水元素，诗意氛围

### 交互风格
- **沉浸式创作**: 减少现代UI元素，专注写作体验
- **渐进式引导**: 像古人拜师学艺一样，逐步掌握创作要领
- **古风反馈**: 用古典语言和意境提供系统反馈

## 🚀 核心特性

### 智能创作伙伴
- **一句话灵感**: 输入想法，AI全程陪伴
- **多场景支持**: 想法、仿写、续写、拆书
- **古风AI**: 理解古典文学语言和意境

### 完整功能设计
- AI自动导演系统
- 智能规划和结构调整
- 角色和世界观管理
- 知识库和RAG能力
- 章节执行和质量控制
- 写法引擎和风格学习

## 📋 项目结构

```
dream-writer/
├── client/              # 前端应用 (React + TypeScript)
│   ├── src/
│   │   ├── components/    # 古风UI组件库
│   │   ├── pages/        # 页面组件
│   │   ├── hooks/        # 自定义hooks
│   │   └── styles/       # 古风主题系统
├── server/              # 后端服务 (Express + TypeScript)
│   └── src/
│       ├── routes/       # API路由
│       ├── services/     # 业务逻辑
│       └── prompting/    # AI提示词管理
├── shared/             # 共享类型定义
└── docs/              # 项目文档
```

## 🎯 使用场景

1. **💭 灵感初现** - 一句话想法，AI引导开篇
2. **📚 仿写经典** - 分析名著结构，模仿笔法
3. **✍️ 续写佳作** - 接力前作，保持风格延续
4. **📖 拆解秘籍** - 分析优秀作品，学习结构技巧
